Detailed Description

template<typename PT, typename... Ts>
struct vgjs::awaitable_tuple< PT, Ts >

Awaitable for awaiting a tuple of vector of type Coro<T>, Function{}, std::function<void(void)>.

The tuple can contain vectors with different types. The caller will then await the completion of the Coros. Afterwards, the return values can be retrieved by calling get().
